# Componentes del sistema

## Componentes UI

- Button: botón reutilizable con variantes
- Card: contenedor visual para información
- Modal: ventana emergente reutilizable

## Componentes de eventos

- EventCard: muestra un evento individual
- EventList: lista de eventos
- EventForm: formulario de creación de eventos

## Componentes de layout

- Layout: estructura general de la aplicación

## Diseño

Todos los componentes están diseñados para ser reutilizables y tipados con TypeScript.
Se utiliza Tailwind CSS para el estilo y la composición de clases.
